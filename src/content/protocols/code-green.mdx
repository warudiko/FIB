## აქტივაციის პირობები

მთავარი ბანკის აღების დროს მოქმედებს პროტოკოლი „კოდი მწვანე“.

## მოთხოვნები / მოქმედებები

1. აბსოლუტურად ყველა თანამშრომელი, რომელიც არ მორიგეობს, არის შვებულებაში ან/და შესვენებაზე, ვალდებულია გამოცხადდეს სამსახურში თავის განყოფილებაში.

2. მთავარი ბანკის გათავისუფლების ხელმძღვანელობას FIB-ის უფროსი პერსონალი იღებს.

3. ყველა აქტიური LSPD პატრული უნდა მივიდეს დაკავების ადგილზე 6-8 წუთის განმავლობაში, დაიწყოს ტერიტორიის დაცვა და გაიყვანოს მშვიდობიანი მოქალაქეები მოქმედების არეალიდან.

4. სხვა სახელმწიფო თანამშრომლები ასევე ვალდებულნი არიან დაუყოვნებლივ მივიდნენ მთავარი ბანკის ტერიტორიაზე და შეასრულონ ყველა ბრძანება მთავარი წამყვანი სპეციალისტისგან.

## სანქციები

პროტოკოლის უგულებელყოფისას სახელმწიფოს სახელმწიფო სტრუქტურა 100 000 დოლარით დაჯარიმდება. პროტოკოლის განმეორებით უგულებელყოფის შემთხვევაში, სტრუქტურა შეიძლება დაისაჯოს სისხლის სამართლის მუხლებით, ასევე ჯარიმა მთელი სტრუქტურისთვის 200 000 აშშ დოლარის ოდენობით.
