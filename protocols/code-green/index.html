კოდი მწვანე

აქტივაციის პირობები

მთავარი ბანკის აღების დროს მოქმედებს პროტოკოლი – „კოდი მწვანე“.

მოთხოვნები / მოქმედებები

  1. ყველა არამორიგე, შვებულებაში მყოფი ან შესვენებაზე მყოფი თანამშრომელი ვალდებულია დაუყოვნებლივ გამოცხადდეს თავის შესაბამის განყოფილებაში.

  2. მთავარი ბანკის გათავისუფლების ოპერაციის ხელმძღვანელობას ახორციელებს FIB-ის უფროსი პერსონალი.

  3. LSPD-ის აქტიური პატრულები ვალდებულნი არიან 6–8 წუთში გამოცხადდნენ ინციდენტის ადგილზე, უზრუნველყონ ტერიტორიის დაცვა და მშვიდობიანი მოქალაქეების ევაკუაცია მოქმედების ზონიდან.

  4. სხვა სახელმწიფო უწყებების თანამშრომლები ვალდებულნი არიან დაუყოვნებლივ გამოცხადდნენ მთავარ ბანკის ტერიტორიაზე და შეასრულონ ოპერაციის ხელმძღვანელის მითითებები.

სანქციები

პროტოკოლის დარღვევის შემთხვევაში სახელმწიფო სტრუქტურას დაეკისრება ჯარიმა შტატის მიერ დადგენილი წესების შესაბამისად.
